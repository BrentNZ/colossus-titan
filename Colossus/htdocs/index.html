Overview

Colossus is a Java clone of Avalon Hill's Titan(tm) boardgame. Right now it allows hotseat play, play versus a working but not very strong AI, and (new) TCP/IP network play. We're mostly working on improving network play and variants right now. Improving the AI will follow.

The readme has more information.

The full docs are here.

Download

If you have Java Web Start installed you can download, install, and play with one click on the Colossus icon:

(Note: Java Web Start will post a security warning, and you'll have to give it permission to access your computer before Colossus will run. This is necessary because Colossus uses files on the local computer for user preferences and save games. If this worries you, feel free to download and inspect the source code.)

You can download the latest version (with source, docs, etc.) from the http download directory. (The executable jar file is no longer standalone.)

Or you can grab the code via Subversion. (This will often get you newer code, so do this if you want to make changes.) See the Subversion instructions on .

(Follow this link for other stuff hosted on SourceForge , like the mailing lists and bug tracker . Note that we don't use every SF feature -- downloads are handled directly from this home page rather than a special file server -- so don't be alarmed if some links go nowhere.)
