README for Colossus ExtTitan Variant

What are ExtTitan and Extended Titan ?

Extended Titan is an extension to regular Titan, the game from Avalon Hill upon which Colossus is based. ExtTitan is a variant pack to allow Colossus player to play Extended Titan. See http://www.cheewai.com/titan/variants_extended.asp, as the original Two Bit Productions page seems unreachable (http://www.2bp.com/). Archive may stille be available from http://www.archive.org/, you can try http://web.archive.org/web/20001008194811/www.2bp.com/titan/extended.html

Frequently Asked Questions

How to use it ?

Just use the "Load Variant" button in Colossus, and open the ExtTitan.var file inside the ExtTitan directory. That should be enough. If not, please report the problem.

What are the differences with regular Titan ?

New creatures, some old creatures are stronger, new recruiting tables, and bigger masterboard. See the page cited above for more details.

What are the differences between Extended Titan and ExtTitan ?

There shouldn't be any now, as we support 8 players, Elementals, ... If you find any difference, please fill a bug report.

Why is the Griffon Rangestriking ?

Guy Dyson (author of Extended Titan) and James Gwinnett (playtester) confirm that they gave griffons rangestriking because wyverns gained slope nativity when they became recruitable in hills, and they wanted to create a balancing advantage for griffons. (There is no real advantage to being native to woods.)
