Release Note for Colossus 0.10.4

Some small new features, plus the usual usability & stability improvements

  • Confirmation dialogs (optional) for omitted/forgotten split, move or recruits (Thanks to David Partidge!).
  • New feature: "Need Lord for Battle Control" (Thanks to Bruce Sears!).
  • New feature: Possibility to mark legions as "skip" for split, move and muster (just made it "somehow", there's room for improvement in the usability)...
  • Player colored angels and captured legion markers.
  • Closing BattleBoard does not kill that player (for not involved player nothing happens at all, for involved player means conceding that battle).
  • Public Game Server rejects now Clients older than 0.10.3.
  • In Concede/Flee and Negotiation dialog, don't allow "x" dialog closing - need to click on of the actions.
  • In WebServer games ( = when client was created by webclient), disable the Suspend button.
  • Numerous small changes to improve reliability / avoid server side warnings/errors/log messages.
  • Changes to building, jarsigning, keystore, local JNLP files, etc.
